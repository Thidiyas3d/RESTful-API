A Restful API for rapid prototyping

Get started

Fire up your browser dev console and paste the code below:

fetch('https://myfakeurl.heroku/posts')
  .then(response =>{
    return response.json()});
  .then(data => console.log(data));

Even better I assure, you won't break the internet.

Resources

Following are the API endpoints.

  • /posts - returns 20 posts objects
  • /users - returns 10 users objects
  • /quotes - returns 5 quotes objects

Routes

Following HTTP methods are supported.

  • GET - /users
  • POST - /users
  • GET - /quotes
  • POST - /quotes
